# Proyecto ABP – Base de Datos CineDB  
Autor:Ezequiel Anaya  
Materia:Base de Datos 1 – IPET 247 – 2025  
Trabajo Práctico N°14

Descripción del Proyecto

Este proyecto implementa la base de datos del sistema **CineDB**, diseñada para gestionar:

- Usuarios del cine  
- Membresías  
- Tipos de comidas  
- Productos del kiosco  
- Salas  
- Películas

El objetivo es construir un modelo relacional completo con 5 o más tablas, relaciones, claves primarias/foráneas, restricciones, inserciones y optimizaciones.
Características Técnicas

-SQL Server (Microsoft SQL Management Studio)  
-Tablas: 6  
-Relaciones: 
  - Usuarios → Membresías  
  - Comidas → Tipos de comidas  
-Restricciones: 
  - PK, FK  
  - NOT NULL  
  - UNIQUE  
  - CHECK  
- Índices para optimización

Archivos del Proyecto

/scripts/schema_database.sql`
Contiene la **creación completa del esquema**, incluyendo:

- Base de datos  
- Tablas  
- Restricciones  
- Índices  

scripts/sample_data.sql
Incluye **datos de ejemplo**, mínimo 5 registros por tabla, como solicita el Trabajo Práctico.
